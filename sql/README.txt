Scripts de Base de Datos PostgreSQL

Contiene la lógica pura de datos del proyecto. Antes de ejecutar la aplicación en Python, se deben cargar estos scripts en pgAdmin 4:

schema_core_final.sql: Creación de tablas y constraints.

inserts_equipo_final.sql: Carga de datos iniciales para pruebas.

query_estadistico.sql: Lógica de negocio para el reporte de clientes.

query_contable.sql: Lógica de flujos de caja e ingresos/egresos.
